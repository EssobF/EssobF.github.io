Veille technologique :

L'intelligence artificielle

Définition :

L’IA désigne la possibilité pour une machine de reproduire des comportements liés aux humains, tels que le raisonnement, la planification et la créativité. Nouvelle étape de la transition numérique mondiale.


Les différents domaines :

Il s'agit d'un Domaine pouvant être séparé en plusieurs parties :
-Le machine learning contenant :
-le deep learning qui contient lui même :
-le modèle discriminatoire
-l'IA générative
-Le LLM (Large Language Model)

Machine learning et deep learning :

Le machine learning signifie que l'on entraine un model d'IA avec des données d'entrainement pour lui permettre d'établir des prédictions avec des données qu'elle n'a jamais vu

On peut qualifier comme étant du deep learning les IA se basant sur le modèle neuronale pour permettre une analyse plus poussées

La différence entre l'IA générative et le modèle discriminatoire :

L'IA générative se base sur des données qui lui ont été soumises dans le but de créer quelque chose de nouveau.

Le modèle discriminatoire se base sur des données qui lui on été soumises dans le but de reconnaitre les types de données

Par exemple, supposons que l'on donnait à ces deux types d'IA des photos d'oranges et des photos de chien. L'ia générative sera capable de générer une image d'orange ou de chien tandis que le modèle discriminatoire sera capable d'établir si des données reçues antérieurement seront des oranges ou bien des chiens

Le LLM (Large Language Model) :

Le LLM désigne les IA qui ont été entrainées sur une large catégorie de donnée avant d'être raffinée pour un objectif précis.

Par exemple, si l'on prenais plusieurs chien ça serait l'équivalent de leur apprendre tous à s'asseoir, donner la pate, etc... puis à leur assigner un rôle plus spécifique comme chien guide, chien de chasse, etc...

Les IA classiques utilisées par la majorité des gens (chatGPT, Gemini et autres...) sont un mix entre l'IA générative et le LLM
